# MultipartFile的API介绍--小鸟程序员

```
1.getInputStream() 返回InputStream读取文件的内容 
2.getOriginalFilename()得到原来的文件名在客户机的文件系统名称 
3.getName() 返回参数的名称 这里返回的也就是 file 
4.isEmpty() 判断是否为空，或者上传的文件是否有内容 
5.getBytes() 将文件内容转化成一个byte[]                   
6.getSize() 返回文件大小 以字节为单位            
7.getContentType() 返回文件的内容类型
```
